# glossary.md — 用語集

## コアドメイン用語

| 用語              | 定義                                                                                      |
|------------------|------------------------------------------------------------------------------------------|
| **フォルダ**       | ユーザーが「開く」操作で選択した、管理対象となるディレクトリ。                              |
| **ファイルエントリ** | 管理対象フォルダの直下に存在する1ファイルを表すドメインオブジェクト。サブフォルダは除く。  |
| **タグ**           | ファイルに付与できるラベル。名前・ID・色を持つ。1ファイルに複数タグを付与できる。           |
| **TagStore**      | タグ定義とファイルとタグの紐付け情報を管理するドメインサービス。`.filetags` に永続化。      |
| **.filetags**     | TagStore の永続化形式。JSON ファイル。管理対象フォルダの直下に配置される。                  |
| **フィルター**     | 選択したタグを持つファイルだけを表示する絞り込み操作。AND/OR は OR 方式。                   |
| **一括選択**       | 複数のファイルエントリを選択状態にする操作。チェックボックス・タグ一括選択が起点になる。     |
| **一括移動**       | 選択中のファイルを、管理対象フォルダ内のサブフォルダへ移動する操作。                        |
| **サブフォルダ**   | 管理対象フォルダの直下に存在するディレクトリ。移動先の候補として使用される。                |

## アーキテクチャ用語

| 用語              | 定義                                                                                      |
|------------------|------------------------------------------------------------------------------------------|
| **ViewModel**    | WPF ビューにバインドされるクラス。CommunityToolkit.Mvvm の `ObservableObject` を継承する。  |
| **Repository**   | TagStore の読み書き（`.filetags` の I/O）を担う層。`ITagRepository` インターフェース経由で使用。 |
| **Command**      | WPF の `ICommand` 実装。ViewModel のメソッドを UI イベントに接続する。                     |
| **ObservableCollection** | WPF データバインディングで変更通知を行うコレクション型。ファイル一覧・タグ一覧に使用。 |

## ファイル状態

| 状態          | 意味                                                     |
|--------------|----------------------------------------------------------|
| **選択中**    | チェックボックスがオンのファイルエントリ。                   |
| **非表示**    | 現在のフィルター条件に合致せず、一覧に表示されていない状態。  |
| **タグなし**  | 1つもタグが付与されていないファイルエントリ。               |
