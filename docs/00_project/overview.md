# プロジェクト概要

> status: active

## アプリケーション名

**FocusKeeper**（フォーカスキーパー）

---

## 目的・背景

PC作業中に「他の作業が気になる」「通知や別タブに目移りする」ことで集中力が途切れる問題を解決する。  
現在取り組んでいる作業をPC画面上に常時表示・管理することで、ユーザーが今やるべきことを忘れず、作業に集中できる環境を提供する。

---

## スコープ

### 対象

- Windowsデスクトップアプリケーション（WPF / .NET 8 / C#）
- シングルユーザー向け（個人の作業集中支援）
- ローカル完結型（インターネット接続不要）

### 対象外

- モバイル・Web・macOS対応
- チーム向けタスク管理・共有機能
- 時間管理・プロジェクト管理（Jira / Trello 連携等）
- AIによるタスク自動生成・提案

---

## 主要機能概要

| 機能 | 説明 |
|------|------|
| 作業オーバーレイ表示 | 現在の作業名をPC画面上に常時表示（他ウィンドウの上に重ねる） |
| 作業の入力・選択 | テキスト入力または履歴・登録済みリストからの選択 |
| 作業リスト管理 | よく使う作業の登録・編集・削除 |
| 設定ウィンドウ | 表示位置・透明度・フォントサイズ・色などのカスタマイズ |
| アクティブウィンドウ監視（任意） | 設定外のアプリに切り替わったとき視覚的に注意を促す |

---

## ターゲットユーザー

- 自宅・オフィスでWindowsを使って作業するナレッジワーカー
- マルチタスクになりがちで、集中力を維持したい人
- ポモドーロ・テクニック等の集中術を補助するツールを求める人

## 成功基準

- 作業名をオーバーレイ表示した状態で、他のアプリを普通に操作できる
- 起動から作業表示まで3秒以内
- 設定ウィンドウから表示位置・スタイルを変更できる
- アプリを終了・再起動しても作業リストと設定が保持される
