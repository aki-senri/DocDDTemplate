# overview.md — プロジェクト概要

## プロジェクト名

**FileTagExplorer**

## 目的・背景

フォルダ内に大量のファイルが存在する場合、ファイル名や日付だけでは整理が困難になる。
FileTagExplorer は、ファイルに自由なタグを付与することで、属性ベースの絞り込みと
一括操作（選択・移動）を可能にし、フォルダ内の整理作業を効率化する。

外部サービスやデータベースに依存せず、管理データはフォルダ直下の `.filetags` ファイルに
保存するため、フォルダを別の場所にコピーしてもタグ情報が維持される。

## 対象ユーザー

- 大量の素材ファイル（画像・動画・ドキュメント）を扱うクリエイター
- プロジェクトファイルを種別・状態で整理したい開発者・研究者

## 主要機能

| 機能            | 説明                                                         |
|----------------|--------------------------------------------------------------|
| フォルダ参照    | フォルダを開き、直下のファイル一覧を表示する                    |
| タグ付与/削除   | ファイルに対してタグを追加・削除する                            |
| タグフィルター  | 選択したタグを持つファイルだけを表示する                        |
| 一括選択        | タグ・チェックボックスで複数ファイルを選択する                  |
| 一括移動        | 選択したファイルをフォルダ内のサブフォルダへ移動する             |
| タグ管理        | タグの作成・名称変更・削除・色設定を行う                        |

## スコープ外

- サブフォルダの再帰的な走査・管理
- ファイルのコピー・削除・リネーム
- クラウドストレージ連携
- 複数フォルダの同時管理
