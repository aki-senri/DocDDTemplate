# 制約条件・前提条件

> status: active

---

## 技術的制約

| 制約 | 内容 | 理由 |
|------|------|------|
| プラットフォーム | Windows専用 | WPF・Windows APIを利用するため |
| ランタイム | .NET 8 | LTSバージョン、2026年11月まで標準サポート |
| UIフレームワーク | WPF | オーバーレイウィンドウの実装に最適（ADR-001） |
| アーキテクチャ | x64のみ | .NET 8の主要ターゲット |
| 最小OS | Windows 10 22H2 | MSIXサポートとWinRT API要件 |
| データベース | SQLite（Microsoft.Data.Sqlite） | ローカルのみ、追加サーバー不要 |
| ネットワーク | 使用しない | ローカル完結型アプリ |

---

## ビジネス制約

| 制約 | 内容 |
|------|------|
| 対応言語 | 日本語のみ（v1.0）。英語対応は v2.0以降に検討 |
| 価格モデル | 無償（v1.0）。将来的な有償プラン導入は未定 |
| ライセンス | MIT（予定）。採用ライブラリのライセンスと競合しないこと |
| 認証・アカウント | 不要。ローカルユーザーデータのみ |

---

## 設計上の前提

- シングルユーザー利用を前提とする（マルチユーザー・共有機能はスコープ外）
- 同一PC上で複数インスタンスの同時起動は想定しない（多重起動防止を実装する）
- オーバーレイウィンドウはDPI対応（高解像度ディスプレイで文字がぼやけない）
- 作業リストの件数上限は1,000件とする（それ以上の場合は警告表示）

---

## 依存関係・外部要件

| 依存 | バージョン | 備考 |
|------|-----------|------|
| .NET Desktop Runtime | 8.x | MSIXにバンドル、個別インストール不要 |
| Windows 10/11 | 22H2以降 | MSIX配布の要件 |
| コード署名証明書 | — | 配布時のみ必要。開発中はテスト証明書を使用 |

---

## 品質上の前提

- CI/CD は GitHub Actions を使用する
- コードレビューはプルリクエスト経由で行う（main への直接 push 禁止）
- invariants.md に定義した不変条件はすべて Roslyn アナライザーで CI 強制する
- テストカバレッジ目標：ViewModel・Service 層 80%以上
