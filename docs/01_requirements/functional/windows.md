# 機能要件（Windows）

> status: active

---

## FR-W01: フォーカス表示機能

### FR-W01-1: オーバーレイウィンドウ表示

- 現在の作業名を画面上に常時表示する
- 他のアプリウィンドウの上に重ねて表示する（最前面固定）
- ウィンドウはタスクバーに表示しない
- ドラッグにより表示位置を移動できる
- 移動後の位置はアプリ終了後も保持される

### FR-W01-2: 表示スタイル

- 背景色・文字色・フォントサイズを設定から変更できる
- 透明度（0%〜90%）を設定から変更できる
- 作業名が設定された幅を超える場合は折り返しまたは省略表示する

### FR-W01-3: オーバーレイの操作

- オーバーレイをクリックしても背面のアプリにフォーカスが移る（クリックスルー）
- オーバーレイ上で右クリックするとコンテキストメニューを表示する
  - 作業変更
  - 作業完了（表示クリア）
  - 設定を開く
  - 終了

---

## FR-W02: 作業入力・選択機能

### FR-W02-1: 新規作業の入力

- テキストフィールドに作業名を自由入力できる
- 入力した作業名をオーバーレイに即座に反映する
- 入力時に作業リスト・履歴からインクリメンタルサーチで候補を表示する

### FR-W02-2: 作業リストからの選択

- 登録済み作業リストを一覧表示し、選択できる
- 作業リストは使用頻度順・最終使用日時順・登録順でソートできる

### FR-W02-3: 作業の完了

- 現在表示中の作業を「完了」としてクリアできる
- 完了した作業は履歴に記録される

---

## FR-W03: 作業リスト管理機能

### FR-W03-1: 作業の登録

- 新規作業を作業リストに登録できる
- 作業名は最大100文字
- 同名の作業は登録できない（バリデーション）

### FR-W03-2: 作業の編集・削除

- 登録済み作業の名称を編集できる
- 登録済み作業を削除できる
- 削除確認ダイアログを表示する

---

## FR-W04: 設定機能

### FR-W04-1: 設定ウィンドウ

- 設定専用のウィンドウを開ける（オーバーレイとは独立）
- 設定ウィンドウはモーダルではなく、他のウィンドウと並存できる

### FR-W04-2: 表示設定

| 設定項目 | 型 | デフォルト | 説明 |
|----------|-----|-----------|------|
| 表示位置X | int | 100 | 画面左端からのピクセル数 |
| 表示位置Y | int | 100 | 画面上端からのピクセル数 |
| 幅 | int | 400 | オーバーレイの幅（px） |
| 透明度 | int | 80 | 背景透明度（%）0〜90 |
| フォントサイズ | int | 16 | テキストのフォントサイズ（pt）|
| 文字色 | string | #FFFFFF | カラーコード |
| 背景色 | string | #1A1A2E | カラーコード |
| 表示モニター | int | 0 | プライマリ=0 |

### FR-W04-3: 動作設定

| 設定項目 | 型 | デフォルト | 説明 |
|----------|-----|-----------|------|
| スタートアップ起動 | bool | false | Windows起動時に自動起動 |
| タスクトレイ常駐 | bool | true | 閉じるボタンでトレイに格納 |
| アクティブウィンドウ監視 | bool | false | 切り替え検出機能のON/OFF |
| 許可アプリリスト | string[] | [] | 監視で無視するアプリ名リスト |

### FR-W04-4: 設定の保存・リセット

- 設定変更は即時または「保存」ボタン押下で適用できる
- 設定を初期値にリセットできる

---

## FR-W05: アクティブウィンドウ監視機能（オプション）

> 設定で有効化した場合のみ動作する

### FR-W05-1: 監視

- Windowsのフォアグラウンドウィンドウを定期的（500ms間隔）に監視する
- 許可アプリリスト外のアプリにフォーカスが移ったことを検出する

### FR-W05-2: アラート

- 検出時にオーバーレイの背景色を変更（アラートカラー）して視覚的に警告する
- 許可アプリに戻ると通常表示に戻る
- アラートカラーは設定から変更できる

---

## FR-W06: タスクトレイ機能

- アプリ最小化時・ウィンドウ閉じ時にタスクトレイに格納する（設定に従う）
- タスクトレイアイコンを右クリックするとコンテキストメニューを表示する
  - 設定を開く
  - 作業変更
  - 終了
- タスクトレイアイコンをダブルクリックすると設定ウィンドウを開く
