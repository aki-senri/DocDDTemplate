# 非機能要件（Windows）

> status: active

---

## NFR-W01: 対応環境

| 項目 | 要件 |
|------|------|
| 対応OS | Windows 10 (22H2以降) / Windows 11 |
| アーキテクチャ | x64 |
| .NET ランタイム | .NET 8 Desktop Runtime（MSIXにバンドル） |
| 最小メモリ | 4GB RAM（推奨 8GB） |
| 最小ストレージ | 100MB（アプリ本体 + データベース） |

---

## NFR-W02: パフォーマンス

| 指標 | 目標値 |
|------|--------|
| 起動時間 | コールドスタートで3秒以内 |
| 作業切り替え応答時間 | 入力確定から表示反映まで200ms以内 |
| 常時CPU使用率 | アイドル時 1%以下、アクティブウィンドウ監視中 3%以下 |
| メモリ使用量 | 常時 100MB以下 |
| DBクエリ応答 | 作業リスト取得（1000件）で100ms以内 |

---

## NFR-W03: 信頼性・安定性

- アプリのクラッシュ時、次回起動時に作業リストと設定が復元できること
- DB書き込みは必ずトランザクションで行い、中断時の破損を防ぐこと
- 異常終了（プロセスキル含む）後も設定ファイルが破損しないこと
- 連続稼働72時間以上でメモリリークが発生しないこと

---

## NFR-W04: ユーザビリティ

- インストール直後にチュートリアルなしで基本操作（作業入力→表示）ができること
- 設定変更はリアルタイムプレビューまたは即時反映で確認できること
- エラーメッセージは原因と対処法をユーザーが理解できる言語で表示すること
- 日本語UI（v1.0スコープ）

---

## NFR-W05: セキュリティ

- アプリはUAC管理者権限なしで動作すること（標準ユーザー権限で実行可能）
- データはローカルユーザーのAppDataフォルダに保存し、他ユーザーからアクセス不可
- 外部ネットワーク通信は行わない（ローカル完結）
- コード署名証明書によるMSIXパッケージへの署名を行う（配布時）

---

## NFR-W06: インストール・配布

| 項目 | 要件 |
|------|------|
| パッケージ形式 | MSIX |
| インストール先 | WindowsAppsフォルダ（MSIX標準） |
| アンインストール | Windowsの「アプリと機能」から完全削除可能 |
| アップデート | MSIX自動更新対応（将来的なMicrosoft Store配布を視野に） |
| 必要権限 | 標準ユーザー権限（UAC不要） |

---

## NFR-W07: アクセシビリティ

- テキストは読み取りやすいコントラスト比（WCAG AA 4.5:1以上）をデフォルトとする
- フォントサイズをユーザーが調整できること（12pt〜36pt）
- Tabキーによるフォーカス移動が設定ウィンドウで機能すること

---

## NFR-W08: オフライン動作

- インターネット接続なしで全機能が利用できること
- 外部APIへの依存は持たないこと

---

## NFR-W09: ログ・診断

- アプリケーションログをローカルファイルに出力する（ローテーション付き）
- ログレベル：INFO / WARN / ERROR
- エラー発生時はスタックトレースをログに含める
- ログ保存先：`%LOCALAPPDATA%\FocusKeeper\Logs\`
- ログローテーション：日次、最大7日分保持
