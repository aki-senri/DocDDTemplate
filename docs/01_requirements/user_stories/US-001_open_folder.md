# US-001 — フォルダを開いてファイル一覧を表示する

## ユーザーストーリー

> **フォルダを整理したいユーザーとして、**  
> フォルダを開いてその直下のファイル一覧を確認したい。  
> **なぜなら、** どのファイルが存在するかを把握してからタグ付け作業を始めたいから。

## 受け入れ条件

### AC-001: フォルダ選択ダイアログでフォルダを開ける

- [ ] ツールバーの「フォルダを開く」ボタンをクリックすると、Windows 標準のフォルダ選択ダイアログが表示される
- [ ] フォルダを選択して「OK」をクリックすると、そのフォルダのファイル一覧が表示される
- [ ] ダイアログをキャンセルした場合、現在の状態は変化しない

### AC-002: ファイル一覧に正しい情報が表示される

- [ ] 一覧には対象フォルダ直下のファイルのみが表示される（サブフォルダ自体は表示しない）
- [ ] 各行にファイル名・ファイルサイズ・更新日時・付与済みタグが表示される
- [ ] `.filetags` ファイル自体はファイル一覧に表示されない
- [ ] ファイルが 0 件の場合は「ファイルが見つかりません」メッセージを表示する

### AC-003: アクセス権限エラーを適切に処理する

- [ ] 読み取り権限のないフォルダを開こうとした場合、エラーダイアログを表示する
- [ ] エラーダイアログには「アクセスが拒否されました」旨のメッセージを含む
- [ ] エラー後も前回開いていたフォルダの状態を維持する
