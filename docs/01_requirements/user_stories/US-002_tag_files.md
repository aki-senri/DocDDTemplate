# US-002 — ファイルにタグを付与・削除する

## ユーザーストーリー

> **ファイルを整理したいユーザーとして、**  
> ファイルにタグを付与・削除したい。  
> **なぜなら、** ファイル名だけでは判断できない属性（状態・種別・優先度など）を  
> ラベルで表現し、後から絞り込めるようにしたいから。

## 受け入れ条件

### AC-004: ファイルにタグを付与できる

- [ ] ファイル行の「タグ追加」ボタン（または右クリックメニュー）から既存タグを選択できる
- [ ] 選択したタグがそのファイルに追加され、タグ列に表示される
- [ ] 1 ファイルに複数のタグを付与できる
- [ ] 付与操作後、`.filetags` ファイルが自動保存される

### AC-005: ファイルからタグを削除できる

- [ ] タグ表示をクリックすると削除確認なしにタグが外れる
- [ ] 削除後、タグ列から該当タグが消える
- [ ] 削除操作後、`.filetags` ファイルが自動保存される

### AC-006: タグを新規作成できる

- [ ] タグ管理パネルから「新しいタグ」ボタンでタグ作成ダイアログが開く
- [ ] タグ名（必須・最大50文字）と色（カラーピッカー、デフォルト: グレー）を設定できる
- [ ] 同名のタグは作成できない（バリデーションエラーを表示）
- [ ] 作成後、タグ一覧に即座に反映される

### AC-007: タグフィルターで絞り込める

- [ ] タグ管理パネルのタグをクリックするとフィルターが ON になり、そのタグを持つファイルだけが表示される
- [ ] 複数タグを選択した場合は OR 条件（いずれかのタグを持つファイルを表示）
- [ ] フィルター中でも「全て表示」ボタンでフィルターを解除できる
- [ ] フィルター状態はフォルダを閉じた際にリセットされる

## UI スケッチ（テキスト）

```
┌─────────────────────────────────┬─────────────────────────┐
│ ファイル一覧                      │ タグ管理                 │
│                                  │ [+ 新しいタグ]           │
│ □ report.pdf  [重要][確認済み]   │ ─────────────────────── │
│ □ draft.docx  [WIP]             │ ● 重要      [フィルター中] │
│ □ image.png                     │ ● WIP                   │
│                                  │ ● 確認済み              │
│                                  │ ─────────────────────── │
│                                  │ [全て表示]               │
└─────────────────────────────────┴─────────────────────────┘
```
