# US-003 — 複数ファイルを選択してサブフォルダへ移動する

## ユーザーストーリー

> **ファイルを整理したいユーザーとして、**  
> タグでフィルターしたファイルを一括で選択し、サブフォルダへ移動したい。  
> **なぜなら、** 同じ属性のファイルをまとめてサブフォルダに仕分けすることで  
> フォルダ構造を整理できるから。

## 受け入れ条件

### AC-008: チェックボックスで複数ファイルを選択できる

- [ ] 各ファイル行の左端にチェックボックスがある
- [ ] ヘッダー行のチェックボックスで表示中の全ファイルを一括選択/解除できる
- [ ] 「タグで一括選択」ドロップダウンから特定タグを持つ全ファイルを選択できる
- [ ] 選択件数がステータスバーに「X 件選択中」として表示される

### AC-009: 選択ファイルをサブフォルダへ移動できる

- [ ] 「移動先」ドロップダウンに現在のフォルダ直下のサブフォルダ一覧が表示される
- [ ] 「移動を実行」ボタンをクリックすると選択ファイルが移動先サブフォルダに移動される
- [ ] 移動先に同名ファイルが存在する場合、上書き確認ダイアログを表示する
- [ ] 移動が完了するとファイル一覧から移動したファイルが消え、選択状態がリセットされる
- [ ] 移動完了後、`.filetags` のエントリから移動したファイルのパスを削除する

### AC-010: 移動を安全に処理する

- [ ] ファイルが 0 件選択の状態で「移動を実行」をクリックしても何も起きない（ボタンは非活性）
- [ ] 移動先サブフォルダが存在しない場合（削除された等）、エラーメッセージを表示する
- [ ] 移動中にエラーが発生したファイルはスキップし、完了後に「N 件失敗」を通知する

## UI スケッチ（テキスト）

```
┌─────────────────────────────────────────────────────────────┐
│ [□ 全選択]  移動先: [archived/ ▼]  [移動を実行]  3件選択中  │
├──────────────────────────────────────────────────────────────┤
│ ☑ report.pdf   1.2 MB  2026-04-10  [重要][確認済み]         │
│ ☑ summary.xlsx 234 KB  2026-04-08  [確認済み]               │
│ □  draft.docx   456 KB  2026-04-15  [WIP]                   │
│ ☑ final.pdf    2.1 MB  2026-04-14  [重要]                   │
└──────────────────────────────────────────────────────────────┘
```

## 移動時のタグデータ処理

移動操作によりファイルが別のサブフォルダに移動した場合：

1. `.filetags` の `files` セクションから該当エントリを削除する
2. 移動先フォルダに `.filetags` がある場合でも、タグの引き継ぎは行わない
   （移動先フォルダは独立した管理スコープのため）
