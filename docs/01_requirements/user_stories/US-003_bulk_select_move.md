# US-003 — 複数ファイルを選択して任意のフォルダへ移動する

## ユーザーストーリー

> **ファイルを整理したいユーザーとして、**  
> タグでフィルターした複数ファイルを一括で選択し、開いたフォルダ配下の任意のフォルダへ移動したい。  
> **なぜなら、** 関連ファイルをまとめて特定のフォルダに集約することで、  
> フォルダ構造を効率よく整理できるから。

## 受け入れ条件

### AC-008: チェックボックスで複数ファイルを選択できる

- [ ] 各ファイル行の左端にチェックボックスがある
- [ ] ヘッダー行のチェックボックスで表示中の全ファイルを一括選択/解除できる
- [ ] 「タグで一括選択」ドロップダウンから特定タグを持つ全ファイルを選択できる
- [ ] 選択件数がステータスバーに「X 件選択中」として表示される

### AC-009: 選択ファイルを任意のフォルダへ移動できる

- [ ] 「移動先を選択」ボタンをクリックすると、フォルダツリーから移動先を選択できる
- [ ] 移動先として選択できるのは、現在開いているフォルダ配下のフォルダのみ（開いたフォルダ自身も含む）
- [ ] 「移動を実行」ボタンをクリックすると選択ファイルが移動先フォルダに移動される
- [ ] 移動先に同名ファイルが存在する場合、上書き確認ダイアログを表示する
- [ ] 移動が完了するとファイル一覧の該当ファイルが新しいパスに更新され、選択状態がリセットされる
- [ ] 移動完了後、`.filetags` 内のキー（相対パス）を移動後のパスに更新する（タグは引き継がれる）

### AC-010: 移動を安全に処理する

- [ ] ファイルが 0 件選択の状態では「移動を実行」ボタンは非活性
- [ ] 移動先フォルダが存在しない場合（削除された等）、エラーメッセージを表示する
- [ ] 移動元と移動先が同じフォルダのファイルはスキップする
- [ ] 移動中にエラーが発生したファイルはスキップし、完了後に「N 件失敗」を通知する

## UI スケッチ（テキスト）

```
┌──────────────────────────────────────────────────────────────────────┐
│ [□ 全選択]  移動先: /archived/2026/  [📁 変更]  [移動を実行]  3件選択中 │
├──────────────────────────────────────────────────────────────────────┤
│ ☑ report.pdf    /           1.2 MB  2026-04-10  [重要][確認済み]    │
│ ☑ summary.xlsx  /drafts/    234 KB  2026-04-08  [確認済み]          │
│ □  draft.docx   /drafts/    456 KB  2026-04-15  [WIP]               │
│ ☑ final.pdf     /           2.1 MB  2026-04-14  [重要]              │
└──────────────────────────────────────────────────────────────────────┘

移動先選択ツリー（[📁 変更] クリック時）:
  ▶ docs/           ← 開いたルートフォルダ
    ▶ drafts/
    ▶ archived/
      ▶ 2026/       ← ここを選択
      ▶ 2025/
    ▶ assets/
```

## 移動時のタグデータ処理

移動操作によりファイルが別のフォルダに移動した場合：

1. `.filetags` の `files` セクションで該当エントリのキー（相対パス）を移動後のパスに更新する
   - 例: `"drafts/report.pdf"` → `"archived/2026/report.pdf"`
2. タグの付与情報は新しいキーに引き継がれる（タグ情報は失われない）
3. 移動先が開いたフォルダの外の場合は操作を拒否する
