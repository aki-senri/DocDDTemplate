# ユーザーストーリー（Windows）

> status: active

---

## US-W01: 作業を画面に表示したい

**As** PC作業をしているユーザー  
**I want** 現在取り組んでいる作業を画面に常時表示したい  
**So that** 作業の途中で何をしていたか忘れず、集中を維持できる

### 受け入れ条件

- [ ] 作業名を入力するとオーバーレイウィンドウに即座に表示される
- [ ] オーバーレイは他のアプリウィンドウの上に表示される
- [ ] 作業名が表示された状態で、背面のアプリを通常通り操作できる（クリックスルー）
- [ ] アプリを再起動しても最後に表示していた作業名が復元される

---

## US-W02: よく使う作業を登録して素早く選びたい

**As** 毎日同じ作業を繰り返すユーザー  
**I want** よく使う作業をリストに登録して、毎回入力せずに選択したい  
**So that** 作業開始時のセットアップ時間を短縮できる

### 受け入れ条件

- [ ] 作業名を作業リストに登録できる
- [ ] 作業選択時にリストが表示され、クリックで選択できる
- [ ] リストは使用頻度順・最終使用日時順に並べ替えられる
- [ ] 登録済み作業を編集・削除できる

---

## US-W03: 作業名の入力候補を絞り込みたい

**As** 多くの作業を登録しているユーザー  
**I want** テキストを入力すると候補が絞り込まれるようにしたい  
**So that** 目的の作業をスクロールなしで素早く見つけられる

### 受け入れ条件

- [ ] 入力フィールドにテキストを入力すると、作業リスト・履歴から前方一致・部分一致で候補が表示される
- [ ] 候補は最大10件表示される
- [ ] 候補からキーボード（↑↓）で選択できる

---

## US-W04: オーバーレイの表示位置を好きな場所に動かしたい

**As** マルチモニター環境のユーザー  
**I want** オーバーレイを画面の邪魔にならない位置に自由に移動したい  
**So that** 作業画面が見えにくくなることなく常時表示できる

### 受け入れ条件

- [ ] オーバーレイをドラッグして任意の位置に移動できる
- [ ] マルチモニター環境でもすべてのモニターに移動できる
- [ ] 移動した位置はアプリ終了後も保持される
- [ ] 設定ウィンドウからX/Y座標で正確な位置を指定できる

---

## US-W05: オーバーレイの見た目をカスタマイズしたい

**As** 自分の好みや作業環境に合わせたいユーザー  
**I want** オーバーレイの色・サイズ・透明度を変更したい  
**So that** 作業の邪魔にならず、かつ見やすい表示にできる

### 受け入れ条件

- [ ] 設定ウィンドウから文字色・背景色をカラーピッカーで変更できる
- [ ] 透明度を0%〜90%の範囲で変更できる
- [ ] フォントサイズを12pt〜36ptの範囲で変更できる
- [ ] 設定変更は即座にオーバーレイに反映される

---

## US-W06: アプリをWindows起動時に自動起動させたい

**As** 毎日FocusKeeperを使うユーザー  
**I want** PCを起動したら自動的にFocusKeeperが起動してほしい  
**So that** 毎回手動で起動する手間を省ける

### 受け入れ条件

- [ ] 設定でスタートアップ起動をONにするとWindows起動時にアプリが起動する
- [ ] スタートアップ起動は設定からOFF（解除）できる
- [ ] 起動時はタスクトレイに格納された状態で起動する（邪魔にならない）

---

## US-W07: 別の作業に気を取られていることに気づきたい

**As** 集中力が散漫になりがちなユーザー  
**I want** 関係ないアプリに切り替えたとき視覚的な警告がほしい  
**So that** 作業から脱線していることに早く気づき、戻ることができる

### 受け入れ条件

- [ ] 設定でアクティブウィンドウ監視をONにできる
- [ ] 許可リスト外のアプリにフォーカスが移るとオーバーレイが警告カラーに変わる
- [ ] 許可リストに戻ると通常カラーに戻る
- [ ] 許可アプリを設定から追加・削除できる

---

## US-W08: アプリを終了せずにバックグラウンドで動かしたい

**As** 常時起動しておきたいユーザー  
**I want** ウィンドウを閉じてもアプリが終了せずタスクトレイに残ってほしい  
**So that** いつでも素早く作業変更や設定変更ができる

### 受け入れ条件

- [ ] ウィンドウの閉じるボタン押下でタスクトレイに格納される（設定でON時）
- [ ] タスクトレイアイコンから設定・作業変更・終了が操作できる
- [ ] 「終了」を選択した場合のみアプリプロセスが終了する
