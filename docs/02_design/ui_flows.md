# 画面遷移・UIフロー

> status: active

---

## 画面一覧

| 画面 | 説明 |
|------|------|
| **オーバーレイウィンドウ** | 現在の作業を常時表示する透明ウィンドウ |
| **作業入力ダイアログ** | 作業名を入力・選択するモーダルダイアログ |
| **設定ウィンドウ** | アプリ動作・表示を設定する独立ウィンドウ |
| **タスクトレイメニュー** | トレイアイコン右クリックで表示するコンテキストメニュー |

---

## アプリ起動フロー

```
アプリ起動
    │
    ├─ 多重起動チェック（Named Mutex）
    │   └─ 既に起動中 → 既存プロセスを前面に → 終了
    │
    ├─ DBマイグレーション
    ├─ 設定ロード
    ├─ 現在の作業ロード（最後の TaskHistory）
    │
    ├─ オーバーレイウィンドウ表示
    ├─ タスクトレイアイコン登録
    │
    └─ アクティブウィンドウ監視開始（設定がONの場合）
```

## オーバーレイウィンドウ

### 通常状態

```
┌─────────────────────────────┐  ← 透明背景（背景色+透明度）
│  ≡  現在の作業名テキスト    │  ← 最前面、クリックスルー
└─────────────────────────────┘
```

- `≡` アイコン: ドラッグハンドル（マウスカーソルを合わせると表示）
- テキストをクリックスルー（背面のアプリにクリックが通る）
- 右クリック: コンテキストメニューを表示

### アラート状態（アクティブウィンドウ監視）

```
┌─────────────────────────────┐  ← アラートカラー背景
│  ⚠  現在の作業名テキスト   │
└─────────────────────────────┘
```

### コンテキストメニュー

```
┌──────────────────┐
│ ✏ 作業を変更     │
│ ✓ 作業を完了     │
│ ─────────────── │
│ ⚙ 設定を開く    │
│ ─────────────── │
│ ✕ 終了           │
└──────────────────┘
```

## 作業入力ダイアログ

オーバーレイの「作業を変更」またはタスクトレイの「作業変更」から開く。

```
┌─────────────────────────────────────────┐
│  作業を選択・入力                  [×] │
├─────────────────────────────────────────┤
│                                         │
│  ┌──────────────────────────────────┐   │
│  │ 作業名を入力...          [×クリア]│   │
│  └──────────────────────────────────┘   │
│                                         │
│  候補リスト（インクリメンタルサーチ）    │
│  ┌──────────────────────────────────┐   │
│  │ ▶ メール対応                     │   │
│  │ ▶ 設計書レビュー                 │   │
│  │ ▶ FocusKeeper開発                │   │
│  │  ...（最大10件）                 │   │
│  └──────────────────────────────────┘   │
│                                         │
│  ─────────────────────────────────────  │
│  [+ 作業リストに登録] [キャンセル] [OK]  │
└─────────────────────────────────────────┘
```

**操作フロー**
1. テキスト入力 → 候補リストが絞り込まれる
2. ↑↓キーで候補選択 → Enterで確定
3. 「OK」押下 → オーバーレイに即座に反映
4. 「+ 作業リストに登録」→ 現在のテキストを作業リストに追加してOK

          │         └───┬──────┬──────────┘           │
