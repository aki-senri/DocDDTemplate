# AIスキル一覧

> FocusKeeperプロジェクト固有のAIスキル定義です。
> 各スキルはセットアップや定型作業を標準化するためのプロンプト集です。

---

## スキル一覧

| スキル | ファイル | 用途 |
|--------|---------|------|
| Windows セットアップ | `setup/windows.md` | 新規開発環境のセットアップ手順をAIに実行させる |

---

## スキルの使い方

スキルファイルの内容をAIへのプロンプトとして使用します。

```
例:
「setup/windows.md のスキルを実行してください」
→ AIが手順に従って開発環境をセットアップします
```

---

## スキル追加ガイドライン

新しいスキルを追加する場合：

1. このREADME の一覧に追加する
2. スキルファイルには以下を含める：
   - 目的・使い所
   - 前提条件
   - 実行手順（ステップバイステップ）
   - 成果物・確認方法
